=== Render Deployment Commands ===

Build Command:
pip install -r requirements.txt && curl -L "https://github.com/newyorkoxygensupply/healix-deploy/releases/download/v1.0/medical_supplies.csv" -o medical_supplies.csv

Start Command:
gunicorn -c gunicorn.conf.py app:app

=== Environment Variables (add in Render dashboard) ===
SITE_URL = https://your-app-name.onrender.com   (update after deploy)
SITE_NAME = Healix
PEXELS_API_KEY = (your Pexels key if you have one)
